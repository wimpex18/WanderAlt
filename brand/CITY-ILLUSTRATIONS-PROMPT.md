# City overview illustrations — design brief for Claude Design

We need three illustrated city plates used as previews in the city-selector
dropdown (see `screenshots/city-selector.png` from the May 2026 product
review). The dropdown shows one card per city; each card has a small SVG
thumbnail on the left, the city name in mono caps on the right, and a
"Live" or "Coming soon" badge.

## Reference style — what to build on

Open `assets/tallinn-overview.svg` (1800×1200 viewBox). It's our existing
hand-illustrated Tallinn plate, generated by `map-world.js`. **Match the
broad direction**: editorial, hand-illustrated quality, warm newsprint
land, muted petrol sea, soft district tints. The big change requested:
**bolder, more colourful, fewer-but-larger landmarks** instead of the
current dense building texture. Think of it as a tourism poster rather
than a survey map.

Notable Tallinn-plate elements to retain in spirit:
- **Warm cream land** (`oklch(0.975 0.014 78)`, ~`#f6f3ec`)
- **Muted petrol sea** (`oklch(0.88 0.018 210)`, ~`#d9e2e3`)
- **Soft green parks** (`oklch(0.86 0.055 140)`, ~`#c5dec2`)
- **Earth-tone roofs** in pinks, mustards, mints, terracotta — saturated
  but desaturated ~40% so the plate reads as illustration, not poster
- **Hand-drawn outline weights** (~2 px main, 1 px detail)
- **No text labels** — these plates are 200 px on screen; type would clash

What to change vs the existing plate:
- **3–5 hero landmarks per city, sized to dominate** (currently Tallinn has
  ~30 small buildings; we want ~5 big ones)
- **More saturated brand accents** — push the petrol `#055959` into a
  signature roof or bay, push lime `#d2dc50` into one signal element
  (sun, awning, flag). Keep the ratio 1:1:8 lime:petrol:everything-else.
- **Sky** — leave white/cream; no horizon line. The plates feel "open".
- **No streets, no neighborhood polygons** — keep the composition clean.

## Output specs

| Property | Value |
|---|---|
| Format | SVG (path-based, no embedded raster, no external fonts) |
| viewBox | `0 0 1800 1200` (matches existing Tallinn plate) |
| preserveAspectRatio | `xMidYMid slice` |
| File path | `assets/tallinn-overview.svg`, `assets/helsinki-overview.svg`, `assets/riga-overview.svg` (overwrite the placeholders) |
| Filesize target | ≤ 50 KB each (Tallinn current is ~110 KB; we can be looser if needed) |
| xmlns | **Required**: `xmlns="http://www.w3.org/2000/svg"` (no xmlns = broken in `<img src>`) |

## Per-city brief

### Tallinn (`assets/tallinn-overview.svg`)
Replace the current dense plate. The current one is a survey-style map;
the new plate should feel like an illustrated postcard.

Hero landmarks (pick 3–5 to render large):
- **Toompea / Pikk Hermann tower** — the medieval skyline above the
  Old Town wall, lime-yellow flag (or red-white Estonian flag)
- **Old Town** — a cluster of pastel gabled roofs (Raekoja plats area)
  with the spire of **Town Hall** rising
- **St Olaf's Church (Oleviste)** — the slim copper-green spire
- **Pirita coastline / TV Tower** — slim concrete spike to the east
- **Telliskivi** — represent the creative quarter with one bold red
  brick warehouse silhouette (this is the WanderAlt natural habitat)

Composition: land curves with the Baltic Sea wrapping the upper-left
edge. Park clusters (Kadriorg, Hirvepark) as soft green blobs.

### Helsinki (`assets/helsinki-overview.svg`)
Currently a "Coming soon" placeholder. Same composition language as
Tallinn — peninsular city, sea on multiple sides.

Hero landmarks:
- **Helsinki Cathedral** — the white neoclassical dome with green
  copper trim, dominating centre
- **Uspenski Cathedral** — red brick with gold onion domes
- **Suomenlinna fortress** — one or two small island silhouettes in
  the sea off the coast
- **Töölö Bay** — soft blue inlet
- **Kallio area** — a cluster of low brick apartment buildings
  (this is where Ääniwalli / Kuudes Linja sit — the underground)

### Riga (`assets/riga-overview.svg`)
Currently a "Coming soon" placeholder. River-city composition — the
Daugava cuts diagonally across the lower portion.

Hero landmarks:
- **House of the Blackheads** — ornate Dutch-renaissance gable, the
  most photographed building in Riga (red & cream, stepped gable)
- **Riga Cathedral (Doms)** — tall brick tower with weathercock
- **Three Brothers** — three narrow gabled houses in a row (Riga's
  oldest civic buildings)
- **Vanšu Bridge** — the white cable-stayed bridge across the Daugava
- **Centrāltirgus** — the iconic Zeppelin-hangar central market
  pavilions (one or two long curved roofs)

The Daugava river curves through; the sea is in the upper-left corner
(Gulf of Riga).

## What NOT to include

- ✗ City name text. The selector UI already labels them.
- ✗ Compass roses, north arrows, scale bars, map legends.
- ✗ Photo-realistic detail. These are illustrations, not maps.
- ✗ Pure black ink — use `#0a0a0c` (our `--c-ink`).
- ✗ Gradients or filter effects. Flat fills only.
- ✗ Tourist labels ("Old Town", "Cathedral"). Silent illustration.
